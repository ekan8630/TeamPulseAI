
# TeamPulse AI

A unique smart task management platform built using:
- React + Vite
- Node.js + Express
- MongoDB

## Run Frontend

cd client
npm install
npm run dev

## Run Backend

cd server
npm install
npm run dev

## Environment Variables

Create .env inside server folder:

PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
